WorldWideWebcam

  Welcome to my first project,  WorldWideWebcam!   I created this webcam viewer in order to view scenes from around the world.  Ive included three viewing options: Random, Category, Random-LIVE only.

Tehcnologies used:
  
  JavaScript
  HTML
  CSS
  AJAX
  api.windy.com
